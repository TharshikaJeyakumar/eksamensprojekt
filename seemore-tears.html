Tears

Year: 2020

Client: School Project

Task: Create CSS Animation

Role: Coding, Drawing & Illustrator

Info:

During my first semester at UCL Erhvervsakademi og Professionhøjskole, Odense, I had a CSS Animation Workshop where our assignment was to make an animation. The main focus was to work with Adobe Illustrator to make a sprint animation. Then we had to code the frames into a CSS Animation using HTML5 & CSS. The intention with this CSS Animation was to make an animation of a girl that cries and then wipes her tears away.
